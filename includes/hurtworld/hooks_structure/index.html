Structure Hooks

CanUseDoubleDoor

void CanUseDoubleDoor(PlayerSession session, DoubleDoorServer door)
{
    Puts("CanUseDoubleDoor works!");
}
  • Called when a player attempts to open a double door
  • Returning true or false overrides default behavior
